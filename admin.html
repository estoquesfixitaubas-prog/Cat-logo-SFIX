🚀 Publicação automática no Netlify (opcional — configurar uma vez)

Se o seu site no Netlify estiver conectado a um repositório do GitHub com "deploy automático" ativado, o painel pode publicar as mudanças direto nesse repositório — o Netlify detecta e atualiza o site sozinho, sem você entrar lá.

Essas informações ficam salvas só neste navegador (nunca são enviadas para mim). Se você não configurar isso, use normalmente o botão "Salvar alterações" abaixo.

Capa da aba Makita

Troque a imagem de capa que aparece ao abrir a aba Makita. Se não escolher um arquivo novo, a capa atual é mantida.

Produtos Makita

Preencha nome, foto e os códigos de cada produto. Use os botões ↑ ↓ para reordenar.

Logos e cabeçalho

Troque as imagens de logo, ajuste o tamanho de cada uma, a posição no cabeçalho e a cor de fundo. Se não escolher um arquivo novo, a logo atual é mantida.

Fundo com marca d'água

É o padrão repetido que aparece atrás do conteúdo da página. Você pode trocar a imagem, ajustar a transparência ou desativar completamente.
